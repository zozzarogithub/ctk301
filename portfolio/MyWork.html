Elemental Glass Cups

My collection of decorative glass cups is themed with 4 main natural elements: Water, Air, Fire, and Earth. These cups are united by a common base shape (a short beer glass profile, with a large rim and concave curve that becomes more exaggerated as it gets closer to the bottom). The glasses vary aesthetically, with each cup using a different set of four colors that are swirled around the entire surfaces. This collection turned in as one project completed the final project requirement for the Glass 3 class at Illinois State University. The project entailed consistently making several shaped glass vessels differentiated by variations in color and pricing them as if I would make them as a full-time job.
